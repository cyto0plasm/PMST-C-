# 💊 PMS — Pharmacy Management System

A desktop application built with **C++** and **CLI WinForms**, designed to streamline pharmacy operations including inventory control, sales, and supplier management.

##  Core Features

- **Pharmacy Management** — Manage pharmacy's inventory and finance
- **User Management** — Role-based access for staff and administrators
- **Product & Category Management** — Full inventory control with categorization
- **Supplier Management** — Track and manage supplier information and orders
- **Cashier Module** — Handle point-of-sale operations efficiently
- **Invoice System** — Generate and manage sales invoices

---

##  Architecture

The project follows the **MVC pattern**, separating concerns across:

- **Models** — Core entities: Pharmacies, Users, Products, Categories, Suppliers, Invoices
- **Views** — WinForms-based UI components
- **Controllers** — Business logic layer handling data flow between models and views
